# Steps to run the code in cmd

* <code>git clone https://github.com/rishavjaiss/fun-with-hooks.git</code>
* <code>cd fun-with-hooks</code>
* <code>code .</code>
* <code>ctrl + ` </code> in windows ,<code>command + ` </code> in Mac
* <code>npm install</code>
* <code>npm start</code>
